* Animating small circles

## Getting Started

Follow these simple steps to run a local copy.

### Installation

1. Clone the repo
```sh
git clone git@github.com:denisraimer/richbee-relation.git
```
2. Install YARN packages
```sh
yarn
```
3. Launch of the project
```JS
yarn run start
```
